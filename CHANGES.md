# Sync Music 2.0 — изменения

- Новый интерфейс приложения и страницы авторизации.
- Фиксированный глобальный плеер.
- Видимый YouTube IFrame Player.
- Живые подсказки поиска.
- История запросов и недавно прослушанных треков.
- Очередь, next/previous, seek и volume.
- Media Session для медиакнопок Windows.
- Улучшенные комнаты, вход по коду и список участников.
- Онлайн-статусы и приглашения друзьям.
- Передача метаданных трека при синхронизации.
- Опциональный Google/YouTube OAuth.
- Скрипты быстрого запуска Windows/Linux.
